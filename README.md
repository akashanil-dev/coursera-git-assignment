# Simple Interest Calculator

A straightforward application to calculate **Simple Interest** based on user-provided principal, rate, and time values.

---

## 📖 What is Simple Interest?

**Simple Interest (SI)** is a quick method to calculate the interest charged on a principal amount over a period of time at a given rate.

### Formula

```
SI = (P × R × T) / 100
```

| Variable | Description |
|----------|-------------|
| **P** | Principal Amount (the initial sum of money) |
| **R** | Rate of Interest (per annum, in %) |
| **T** | Time Period (in years) |

**Total Amount:**
```
A = P + SI
```

---

## ✨ Features

- Calculate simple interest instantly
- Displays the total amount payable/receivable
- Clean and easy-to-use interface
- Input validation to prevent incorrect calculations

## 🛠️ How It Works

1. The user enters the **Principal Amount**, **Rate of Interest**, and **Time Period**.
2. On clicking **Calculate**, the app computes:
   - Simple Interest using `SI = (P × R × T) / 100`
   - Total Amount using `A = P + SI`
3. The results are displayed instantly on the screen.
